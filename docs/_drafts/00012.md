## 始めに

今までWindowsを使用していましたが、Zorinに引っ越してみましたので、まずはキャプチャをお見せします。

Windowsの主な不満としてWindowsUpdateが重いこと（累積更新）やメモリ消費が多いこと（常駐サービスや仮想化）がありました。
そこでWindowsライクなUIを提供するLinuxとして有名なZorinをインストールしてみました。
このOSはWindowsユーザがLinuxに移行しやすいように設計されたディストリビューションです。

この記事ではOS本体のインストール後の初期設定や、おすすめのアプリケーションについて紹介します。
なお、OS本体のインストール手順は画面を見ながら進めるだけなので割愛します。

ちなみに自分は仕事やプライベートでCLIのUbuntuやDebianを使用しているのですが、単純にインストールするだけあれば敷居は高いとは感じませんでした。
Linux初心者でも問題なくインストールできると思います。
しかし、この記事では紹介しませんが、パーティションやブートローダの設定などは初心者には難しいと感じました。
こちらは別の記事で紹介する予定です。

## 想定する読者

- WindowsユーザでLinuxに興味がある方
- WindowsユーザでWindowsに不満がある方
- LinuxユーザでZorinを試してみたい方
- LinuxユーザでZorinのおすすめアプリを知りたい方

## PC構成

WindowsとZorinをデュアルブートでインストールしました。

- OS:
  - Windows 11 Pro
  - Zorin OS 18.1 Core
- CPU: Intel i7-8700 (12) @ 4.600GHz
- GPU: NVIDIA GeForce RTX 4060
- SSD:
  - SATA: 480GB (Windows用)
  - M.2 NVMe: 512GB (Zorin用)
- Memory: 16GB
- Monitor: FHD x 3

## ログイン手順

### ウィンドウシステム: Wayland

ログイン時にWaylandを選択します。

ZorinにはウィンドウシステムとしてWaylandとX11の2種類ありますが、Waylandを選択します。
Chrome（TwitterやChatGPTなど）での描画効率が良いこと、入力遅延がほぼないことが理由です。
ただし、GPUがNVIDIAなので相性が完全ではないこと、一部アプリがWaylandに非対応であることには注意が必要です。

## 構築手順

### 外観設定

#### フォント: Noto Sans CJK / Cascadia Code

メニューやドキュメントの表示に適した日本語フォントと、コードやターミナルの表示に適したフォントをインストールしました。

```shell
sudo apt install fonts-noto fonts-noto-cjk fonts-cascadia-code
```

設定操作:

- Zorin Appearance ＞ フォント

| 項目             | 設定値                   |
| ---------------- | ------------------------ |
| インターフェース | Noto Sans CJK JP Regular |
| 等幅             | Cascadia Code Roman      |
| ドキュメント     | Noto Sans CJK JP Regular |

#### テーマ: Materia

標準のダークテーマだと暗すぎる（非アクティブ時の画面が見づらい）ので、ダークテーマの中でも少し明るいテーマをインストールしました。

```shell
sudo apt install materia-gtk-theme
```

設定操作:

- Zorin Appearance ＞ テーマ ＞ その他

| 項目             | 設定値       |
| ---------------- | ------------ |
| アプリケーション | Materia-dark |

#### ディレクトリ: 英語化

標準だとユーザホームのディレクトリ名が日本語（例: ドキュメント）になっているため、英語化しました。

```shell
LANG=C xdg-user-dirs-gtk-update
rm -rf ~/.cache/*
```

実施内容:

- 和名の旧フォルダ削除
- 再ログイン（GNOMEセッション更新）

懸念事項:

この対応によりログイン時のシェル起動に失敗する可能性があります。
それにより、再度パスワードを入力する必要があるかもしれません。

#### 標準のショートカット

設定操作:

- 設定 ＞ キーボード ＞ キーボードショートカット

| 項目                                 | キー |
| ------------------------------------ | ---- |
| ウィンドウ ＞ ウィンドウを最大化する | 無効 |
| ウィンドウ ＞ ウィンドウを戻す       | 無効 |
| ウィンドウ ＞ 画面右半分に表示する   | 無効 |
| ウィンドウ ＞ 画面左半分に表示する   | 無効 |
| システム ＞ 通知リストを表示する     | 無効 |

#### 独自のショートカット

後続の「クリップボード: GPaste」でインストールするアプリのショートカット設定です。
Windowsのショートカットと似た挙動を実現させます。

設定操作:

- 設定 ＞ キーボード ＞ キーボードショートカット ＞ 独自のショートカット

| 名前   | コマンド         | キー    |
| ------ | ---------------- | ------- |
| GPaste | gpaste-client ui | Super+V |

### 入出力デバイス接続

#### デバイス接続: Bluetooth接続

PCにBluetoothレシーバーをUSB接続していましたので、Bluetooth接続を有効にしました。

```shell
bluetoothctl
power on
agent on # デバイスからのペアリングリクエストを受信するため、エージェントを起動
default-agent # デフォルトのエージェントを有効にし、ペアリング時の認証（Passkeyの入力など）を自動化
```

#### マウス設定: Solaar

LogiのBoltやUnifyingレシーバーでデバイスを接続しているため、
Windowsでは専用の管理アプリ（Logi Options+）を使用していましたが、Linuxでは提供されていないため、代替アプリをインストールしました。

```shell
sudo apt install solaar
```

### システム管理・保守

#### システムモニター: Mission Center

システムモニターはGNOME標準のアプリがありますが、より詳細な情報を表示できるアプリをインストールしました。
GNOME標準のアプリでは提供されていないのですが、GPUの使用率や温度を表示できます。

なお、aptやsnapでは提供されていないため、flatpakでインストールしました。

```shell
flatpak install flathub io.missioncenter.MissionCenter
```

#### バックアップ: Timeshift

バックアップはGNOME標準のアプリがありますが、システムの状態を定期的に保存して、必要に応じて復元できるアプリをインストールしました。
GNOME標準のアプリより、システム更新や設定変更によるトラブルから復旧しやすいです。

```shell
sudo apt install timeshift
```

#### ディスク管理: GParted

ディスク管理はGNOME標準のアプリがありますが、パーティションの編集に特化したアプリをインストールしました。
GNOME標準のアプリより、パーティションの作成・削除・リサイズやファイルシステムの変更などの一般的な操作を細かく柔軟に行えます。

```shell
sudo apt install gparted
```

#### セキュリティ: ファイアウォール

インストールはされていますが、デフォルトでは無効になっているため、有効にします。

設定操作:

- ファイアウォール設定ツール

| 項目   | 設定値 |
| ------ | ------ |
| Status | ON     |

### 日常利用アプリ

#### クリップボードマネージャ: GPaste

GNOME標準のクリップボードマネージャは履歴を保存できないため、履歴を保存できるアプリをインストールしました。
なお、GNOMEシェル拡張（GPaste）をインストールすることで、ログイン時に自動起動されますが、
インストール後はログオフ＆ログインが必要です。

```shell
sudo apt install gpaste-2
sudo apt install gnome-shell-extension-gpaste
```

設定操作:

- GPaste の設定 ＞ 挙動全般

| 項目                             | 設定値 |
| -------------------------------- | ------ |
| GNOME shell 拡張機能を有効化する | ON     |

#### アーカイブマネージャ: File Roller（標準）

GNOME標準のアーカイブマネージャは一般的な圧縮・解凍の操作を行うのに十分な機能を提供しているため、
特に追加のアプリはインストールしません。

#### コミックビューア: MComix

WindowsではHoneyviewを使用していましたが、Linuxでは提供されていないため、代替アプリをインストールしました。
紙の漫画のように読めるアプリはこれ一択だと思います。

```shell
sudo apt install mcomix
```

設定操作:

- 表示

| 項目                 | 設定値 |
| -------------------- | ------ |
| 見開き表示           | ON     |
| 漫画のページ順に表示 | ON     |

#### 動画再生: mpv

WindowsではMPC-HCもしくはMPC-BEを使用していましたが、Linuxでは提供されていないため、代替アプリをインストールしました。

```shell
sudo apt install mpv
```

動作は非常に軽くて使いやすいのですが、このアプリ単体だとGUIによる操作がほとんど提供されていないため、
プレイリスト管理、ABリピート、シークバーでのサムネ表示などはスクリプトを別途実装もしくは導入する必要があります。
下記自作リポジトリのスクリプトを導入して、上記の機能を実現しています。

- [https://github.com/silverag-corgi/mpv-config](https://github.com/silverag-corgi/mpv-config)

#### サムネイル生成: ffmpegthumbnailer

ファイルマネージャーで動画ファイルのサムネイルを表示するアプリをインストールしました。

```shell
sudo apt install ffmpegthumbnailer
```

#### ブラウザ: Chrome

```shell
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo apt install ./google-chrome-stable_current_amd64.deb
sudo apt update
sudo apt install google-chrome-stable
```

拡張機能:

- AutoScroll（自動スクロール）
  - Solaarではホイールクリックに割り当て不可能
- crxMouse（マウスジェスチャー）
