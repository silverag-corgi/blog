# ブログの運用方針

このブログは以下の方針で運用します。

  - 自分がしたことや考えたことなどを備忘録として記録する
  - 記事は未来の自分に対して発信し、次点で不特定多数の誰かに発信する

また、カテゴリごとに記事を以下のように作成しています。

<div class="scroll-table">

| カテゴリ     | 記事内容                                                                         | 語尾       | 
| ------------ | -------------------------------------------------------------------------------- | ---------- | 
| 技術日記     | こんなことやってみました                                                         | ですます調 | 
| 技術紹介     | こんな技術があります<br>こんな手順で実現できます<br>こんなルールで運用しています | だである調 | 
| (ブログ全体) |                                                                                  | ですます調 | 

</div>
